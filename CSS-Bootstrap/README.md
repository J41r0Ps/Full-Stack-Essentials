- CSS selectors, variables, nesting
- Forms and styling forms
- Bootstrap theming and customization

## Topics

1. Bootstrap 5.3
   - Introduction
   - Components

2. CSS
   - Selectors
   - Variables
   - Nesting

3. Forms
   - Form elements
   - Styling forms

4. Theming
   - Customize Bootstrap

Each subfolder contains exercises with accompanying HTML, CSS, and JS where needed.
